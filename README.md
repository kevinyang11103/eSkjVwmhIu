# 高校教室管理系统 python904

## 项目概述

高校教室管理系统是一个基于Python语言的Web应用程序，旨在帮助高校更有效地管理教室资源，提高教室利用率。

## 技术栈

- **后端**：Python 3.7，Django框架
- **前端**：Vue.js
- **数据库**：MySQL
- **开发工具**：PyCharm

## 功能模块

- **查看首页**：展示系统概览和通知。
- **个人中心**：用户可以查看和编辑个人信息。
- **教师管理**：管理系统中的教师账户和相关信息。
- **学生管理**：管理系统中的学生账户和相关信息。
- **教室信息管理**：维护教室的基本信息和状态。
- **教师申请管理**：处理教师对教室使用的申请。
- **学生申请管理**：处理学生对教室使用的申请。
- **课时表管理**：编辑和管理课时表信息。
- **教师取消预约管理**：记录和管理教师的预约取消请求。
- **学生取消预约管理**：记录和管理学生的预约取消请求。

## 系统角色

系统中包含以下角色：
- 系统管理员：负责整个系统的管理和维护。
- 教师：可以申请使用教室，管理自己的课程。
- 学生：可以申请使用教室。

## 运行环境

- Python 3.7+
- MySQL数据库
- Node.js环境（前端Vue.js）
